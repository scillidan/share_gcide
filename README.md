# share_gcide

[![Create Releases](https://github.com/scillidan/share_gcide/actions/workflows/releases.yml/badge.svg)](https://github.com/scillidan/share_gcide/actions/workflows/releases.yml)

Data from `gcide.zip` (StarDict) on [Free StarDict dictionaries](https://tuxor1337.frama.io/firedict/dictionaries.html) ([0.48](https://dict.org/bin/Dict?Form=Dict3&Database=gcide)?). I can't process the data in [version 0.54](https://gcide.gnu.org.ua/download) yet.

Only one obvious style repair was made. See preview screenshot [here](temp/asset).

## Usage

1. Download files from [Releases](https://github.com/scillidan/share_gcide/releases).
2. Use them in GoldenDict (StarDict), Yomichan/Yomitan.
3. See preview screenshot [here](temp/asset/).
