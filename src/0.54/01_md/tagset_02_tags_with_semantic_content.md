## Tags with Semantic Content

| Tag           | Font   | Meaning and Description                                                                                         |
| :-            | :-     | :-                                                                                                              |
| `<altsp>`     | *      | Alternative spelling segment. Almost always contained within square brackets after the main definition segment. |
| `<ant>`       | italic | Antonym.                                                                                                        |
| `<asp>`       | italic | Alternative spelling. The actual word which is an alternative spelling to the headword.                         |
| `<au>`        | italic | Authority or author. Used where an authority is given for a definition...                                       |
| `<bio>`       | *      | Marks a biography. Should be longer than a short mention of who a person was...                                 |
| `<biography>` | *      | Same as `<bio>`                                                                                                 |
| `<booki>`     | italic | Marks the name of a book, pamphlet, or similar document.                                                        |
| `<branchof>`  | *      | A field of knowledge which of which the headword is a division.                                                 |
| `<caption>`   | *      | Caption of a figure or table.                                                                                   |
| `<cas>`       | *      | tags the CAS (Chemical Abstracts Service) registry number for a chemical substance.                             |
| `<causes>`    | italic | tags the infectious disease caused by the headword.                                                             |
| `<causesp>`   | *      | Same as `<causes>` without the italic type.                                                                     |
| `<causedbyp>` | *      | Same as `<causedby>` without the italic type.                                                                   |
| `<causedby>`  | italic | inverse of `<causes>`...                                                                                        |
| `<centered>`  |        | Used only for the single letter in the headers to each letter of the alphabet.                                  |
| `<city>`      | *      | marks the proper name of a city. Used only occasionally and not consistently at this stage.                     |
| `<cnvto>`     | italic | Converted to: used to tag substances which are products prepared by conversion from the headword.               |
| `<colheads>`  | *      | List of heads for the columns of a table.                                                                       |
| `<coltitle>`  | *      | Title of a column in a table.                                                                                   |
| `<comm>`      | *      | Comment -- differs from `<note>` in being in-line with the definition paragraph.                                |
| `<company>`   | *      | Name of a company (commercial firm). Compare `<org>`.                                                           |
| `<compof>`    | italic | Composed of. Tags a substance of which the headword is at least partly composed.                                |
| `<contains>`  | *      | marks an object contained within the headword.                                                                  |
| `<contr>`     | italic | Contrasting word.                                                                                               |
| `<country>`   | *      | Name of a country (nation) of the world.                                                                        |
| `<cref>`      | italic | Collocation reference.                                                                                          |
| `<date>`      | *      | A Date, of any type, e.g. `<date>Dec. 25</date>`.                                                               |
| `<datey>`     | *      | Date-with-year tags a date containing a year.                                                                   |
| `<def>`       | *      | A definition. The definition may have subfields...                                                              |
| `<divof>`     | *      | Division of the headword...                                                                                     |
| `<edi>`       | *      | Marks an education institution, a subtype of organization.                                                      |
| `<emits>`     | *      | Tags a physical object or form of radiation emitted by the headword.                                            |
| `<figure>`    |        | Just a place-holder for illustrations, but seldom used.                                                         |
| `<film>`      | italic | Marks the name of a movie film.                                                                                 |
| `<fld>`       | italic | Field of specialization. Most often used for Zoology and Botany...                                              |
| `<geog>`      | *      | Name of a geographical region of any size; if applicable, the more specific `<city>`, `<state>`, or `<country>` |
| `<hypen>`     | *      | Hypernym.                                                                                                       |
| `<illu>`      | *      | Illustrative usage...                                                                                           |
| `<illust>`    | *      | Illustration place-holder. Seldom used.                                                                         |
| `<img>`       | *      | HTML usage -- points to an image file...                                                                        |
| `<intensi>`   | *      | Points to a word whose meaning is an intensified form of the headword.                                          |
| `<item>`      | *      | Designates one item in a row of a table...                                                                      |
| `<itran>`     | italic | Translation into a foreign (non-English) language...                                                            |
| `<itrans>`    | italic | Same as `<itran>`                                                                                               |
| `<jour>`      | *      | Title of a journal (periodical).                                                                                |
| `<matrix>`    | *      | Always a filled rectangular array.                                                                              |
| `<matrix2x5>` | *      | A 2x5 matrix (2 rows by 5 columns).                                                                             |
| `<mstypec>`   | *      | Multiple synonymous subtypes.                                                                                   |
| `<mtable>`    | *      | Multiple table, encloses `<table>` figures.                                                                     |
| `<musfig>`    | *      | Music figure. Only in a note under the entry "Figure"...                                                        |
| `<p>`         | *      | Paragraph tag, used always in pairs.                                                                            |
| `<person>`    | *      | Marks the proper name of a person.                                                                              |
| `<persfn>`    | *      | Marks the name of a person, when only one name (usually the last name) is given.                                |
| `<publ>`      | *      | Marks the name of a publication other than book, which is marked by `<booki>`.                                  |
| `<qpers>`     | *      | Tags the name of a person who is speaking, within a quotation.                                                  |
| `<qperson>`   |        | Same as `<qpers>`                                                                                               |
| `<cp>`        | *      | Collocation, plain text -- used to tag phrases that should be parsed as a unit...                               |
| `<qau>`       | italic | Always right-justified, as described for `<au>`.                                                                |
| `<ref>`       | *      | A reference to a word in the vocabulary.                                                                        |
| `<refs>`      | *      | Marks the set of references used for a longer article...                                                        |
| `<river>`     | *      | Marks the name of a river -- a proper name.                                                                     |
| `<rj>`        | *      | Right justified.                                                                                                |
| `<row>`       | *      | Designates a row in a table.                                                                                    |
| `<state>`     | *      | Name of a geopolitical state...                                                                                 |
| `<subtypes>`  | *      | Lists subtypes of the headword.                                                                                 |
| `<sup>`       | *      | Superscript                                                                                                     |
| `<supr>`      | *      | Supra...                                                                                                        |
| `<table>`     | *      | Always a filled rectangular array, having `<row>` and `<item>` elements.                                        |
| `<td>`        | *      | Table datum - one cell in a table.                                                                              |
| `<th>`        | *      | Table header.                                                                                                   |
| `<tradename>` | *      | Tags a commercial Trade name.                                                                                   |
| `<ttitle>`    | *      | Table title (Larger than normal font).                                                                          |
