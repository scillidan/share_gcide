# Functional Tags

| Tag           | Font                      | Meaning and Description                                                                      |
| :-            | :-                        | :-                                                                                           |
| `<--    -->`  | *                         | Comment, not a tag. These segments should be deleted from the written or printed text.       |
| `<!   !>`     | *                         | A comment. Used to indicate page numbers in the public domain version.                       |
| `<abbr>`      | italic                    | Tag for abbreviations, when mentioned within the definition text.                            |
| `<adjf>`      | small caps                | Tags for the actual adjective or adverb comparatives or superlatives.                        |
| `<altname>`   | italic                    | Alternative name. Usually for plants or animals...                                           |
| `<altnpluf>`  | italic                    | Same as `<altname>`...                                                                       |
| `<amorph>`    | *                         | Adjective morphological segment...                                                           |
| `<as>`        | *                         | A segment occurring within the definitional sentence...                                      |
| `<cd>`        | smaller                   | Collocation definition...                                                                    |
| `<col>`       | bold, smaller             | Collocation. A word combination containing the headword...                                   |
| `<colp>`      |                           | Collocation, no typographic significance.                                                    |
| `<conjf>`     | small caps                | The conjugated (non-infinitive) forms of verbs...                                            |
| `<cs>`        | smaller                   | Collocation segment. The font and size is normal in a cs...                                  |
| `<decf>`      | small caps                | Declension form. The actual morphological variants of nouns or pronouns...                   |
| `<ecol>`      | *                         | Embedded Collocation...                                                                      |
| `<ent>`       | Bold                      | Entry field. Gives the headword without accent or syllabication marks...                     |
| `<er>`        | small caps                | Entry reference...                                                                           |
| `<ety>`       | *                         | Etymology. Always contained within square brackets...                                        |
| `<ets>`       | italic                    | Etymological source...                                                                       |
| `<etsep>`     | italic                    | Etymological source, being the name of a person or geographical location...                  |
| `<ex>`        | italic                    | Example. An example of usage of the headword...                                              |
| `<fr>`        | *                         | Frequency of use, ordinal rank...                                                            |
| `<fu>`        | *                         | First use. A date at or around which the first use of this word...                           |
| `<grk>`       |                           | Greek transliteration.                                                                       |
| `<hw>`        | bold, larger              | A headword. Each main entry begins with the `<hw>` mark, and ends at the next `<hw>` mark... |
| `<mark>`      | italic, but               | Usage mark. Almost always within square brackets...                                          |
| `<markp>`     | *                         | A usage mark in plain type (not italic)....                                                  |
| `<mcol>`      | *                         | Multiple collocation...                                                                      |
| `<mhw>`       | *                         | Multiple headword...                                                                         |
| `<nmorph>`    | *                         | Noun morphology section...                                                                   |
| `<note>`      | *                         | Explanatory note...                                                                          |
| `<plu>`       | *                         | Plural. The "plural" segment starts with a "pl."                                             |
| `<pos>`       | italic                    | Part of speech. Always an abbreviation...                                                    |
| `<epos>`      | *                         | Part of speech, referring to words in etymologies, normal type...                            |
| `<plw>`       | small caps                | Plural word...                                                                               |
| `<pr>`        | *                         | Pronunciation. The default font is normal...                                                 |
| `<q>`         | smaller by                | Quotation...                                                                                 |
| `<qau>`       | italic, right             | Quotation author.                                                                            |
| `<qex>`       | italic                    | Quotation example...                                                                         |
| `<sd>`        | italic                    | Subdefinition...                                                                             |
| `<ship>`      | italic                    | The name of a ship...                                                                        |
| `<sing>`      | *                         | Singular. Analogous to the `<plu>` segment...                                                |
| `<singw>`     | small caps                | Singular word.                                                                               |
| `<sn>`        | bold, larger              | Sense number...                                                                              |
| `<source>`    | italic                    | Source...                                                                                    |
| `<syn>`       | plain                     | Synonyms...                                                                                  |
| `<usage>`     | narrower                  | Comparisons of word usage...                                                                 |
| `<ver>`       | *                         | Verified for current accuracy...                                                             |
| `<vmorph>`    | *                         | Verb morphology (conjugation) segment...                                                     |
| `<wordforms>` | *                         | Morphological derivatives...                                                                 |
| `<wf>`        | bold, larger              | Same font as `<hw>` marks the actual morphological forms...                                  |
| `<def2>`      | *                         | Second definition...                                                                         |
| `<specif>`    | *                         | "Specifically:" Used to mark the words...                                                    |
| `<pluf>`      | italic                    | Plural form. Used exclusively to mark...                                                     |
| `<uex>`       | italic                    | Usage example...                                                                             |
| `<isa>`       | italic                    | Supertype (hypernym)...                                                                      |
| `<chform>`    | plain, subscript          | Chemical formula.                                                                            |
| `<chformi>`   | plain                     | Chemical formula same as `<chform>`...                                                       |
| `<chname>`    | *                         | Chemical name.                                                                               |
| `<see>`       | *                         | A "see" reference....                                                                        |
| `<mathex>`    | italic                    | Mathematical expression...                                                                   |
| `<ratio>`     | italic                    | Also a mathematical expression...                                                            |
| `<singf>`     | italic                    | Singular form...                                                                             |
| `<mord>`      | *                         | Morphological derivation.                                                                    |
| `<fract>`     | a stack,                  | Fraction. Used for non-numerical fractions...                                                |
| `<exp>`       | superscript, smaller font | Exponential. Used in mathematical expressions.                                               |
| `<xlati>`     | italic                    | Translation (e.g. of Greek)...                                                               |
| `<tran>`      | italic                    | Word translated: the word in italic is translated by a subsequent word...                    |
| `<tr>`        | italic                    | Translation of the preceding word...                                                         |
| `<fexp>`      | *                         | Functional expression (math)...                                                              |
| `<iref>`      | italic                    | Illustration reference...                                                                    |
| `<figref>`    | italic                    | Figure reference...                                                                          |
| `<figcap>`    | *                         | Figure caption...                                                                            |
| `<figtitle>`  | *                         | Figure title...                                                                              |
| `<funct>`     | *                         | Tags a mathematical function or expression...                                                |
| `<chreact>`   | *                         | Chemical reaction...                                                                         |
| `<ptcl>`      | italic                    | Verb Particle...                                                                             |
| `<tabtitle>`  |                           | Table Title...                                                                               |
| `<title>`     | italic                    | Title of a literary work...                                                                  |
| `<root>`      | *                         | Square root...                                                                               |
| `<vinc>`      | *                         | Vinculum...                                                                                  |
| `<nul>`       | plain                     | Nultype...                                                                                   |
| `<cd2>`       | *                         | Second collocation definition...                                                             |
| `<hypen>`     | *                         | Hypernym...                                                                                  |
| `<stype>`     | italic                    | Subtype...                                                                                   |
| `<headword>`  | *                         | Subtype...                                                                                   |
| `<simto>`     | *                         | Similar-to...                                                                                |
| `<conseq>`    | *                         | Consequence...                                                                               |
| `<hascons>`   | *                         |                                                                                              |
| `<consof>`    | *                         | Consequence of...                                                                            |
| `<part>`      | italic                    | Part...                                                                                      |
| `<parts>`     | italic                    | Part, plural form...                                                                         |
| `<partof>`    | *                         | Marks a word designating something...                                                        |
| `<contxt>`    | *                         | Context...                                                                                   |
| `<grp>`       | *                         | Marks the name of a group...                                                                 |
| `<membof>`    | italic                    | Marks a group of which the headword is a member...                                           |
| `<member>`    | italic                    | Marks a member of a group defined by the headword...                                         |
| `<members>`   | italic                    | Same as `<member>`, but marks a plural word...                                               |
| `<method>`    | *                         | Designates a special type of definition...                                                   |
| `<corpn>`     | *                         | Name of a business company...                                                                |
| `<corr>`      | italic                    | Correlative...                                                                               |
| `<qperson>`   | italic                    | Marks the name of a person...                                                                |
| `<org>`       | *                         | Marks the name of an organization...                                                         |
| `<prod>`      | italic                    | produces...                                                                                  |
| `<prodp>`     | *                         | produces (plainfont)...                                                                      |
| `<prodby>`    | *                         | produced by...                                                                               |
| `<prodmac>`   | italic                    | produces...                                                                                  |
| `<stage>`     | italic                    | life stage of an organism...                                                                 |
| `<stageof>`   | *                         | An organism one of whose life stages...                                                      |
| `<inv>`       | italic                    | Inversely related to headword...                                                             |
| `<methodfor>` | italic                    | Is a method to...                                                                            |
| `<examp>`     | italic                    | Example or instance of...                                                                    |
| `<sfield>`    | *                         | Subfield of the headword...                                                                  |
| `<stage>`     | italic                    | A stage of life of the headword...                                                           |
| `<unit>`      | italic                    | A unit of measure...                                                                         |
| `<uses>`      | italic                    | Tags a tool or method used by the headword...                                                |
| `<usedfor>`   | *                         | Tags a method or process...                                                                  |
| `<usedby>`    | italic                    | Tags a tool or method which uses the headword...                                             |
| `<perf>`      | italic                    | performs...                                                                                  |
| `<recipr>`    | italic                    | reciprocal...                                                                                |
| `<sig>`       | italic                    | significance, meaning...                                                                     |
| `<wns>`       | italic                    | WordNet sense...                                                                             |
| `<w16ns>`     | italic                    | WordNet version 1.6 sense...                                                                 |
| `<wnote>`     | *                         | A note related to usage...                                                                   |
