## Aspirants

```
'a  -- ἀ
"a  -- ἁ
```

## Accents

```
a`  -- ά
a~  -- ὰ
a^  -- ᾶ
```

```
'a`  -- ἄ
'a~  -- ἂ
'a^  -- ἆ
"a`  -- ἄ
"a~  -- ἂ,
"a~  -- ἃ
```
   
## Iota subscriptum

```
a`,   -- ᾴ
'a`,  -- ᾄ
```

## Diaeresis

```
i:   -- ϊ
i:^  -- ῗ
i:`  -- ῒ
```
